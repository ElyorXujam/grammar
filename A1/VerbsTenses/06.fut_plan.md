### **Going to (Plans & Predictions)**

**Structure**
am / is / are + going to + Base Verb

* Plus: We're going to update the system next week.
* Minus: He isn't going to attend the call.
* Ask: Are you going to join us?

**When to use it**
1. Pre-made plans: Decisions made before speaking.
2. Clear evidence: The server load is 99%, it's going to crash.

**Noticeable Feedback**
* Writing vs. Speaking: It sounds like "gonna" when speaking. Never write "gonna" in documentation or professional chats.
* Missing "to be": Incorrect: I going to fix it -> Correct: I am going to fix it
