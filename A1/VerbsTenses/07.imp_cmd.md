### **Imperatives (Direct Commands)**

**Structure**
No pronouns/subjects (I, you, he). Start directly with the Base Verb.

* Do this: Review this file. / Come here.
* Don't do this: Don't touch that script. / Don't panic.

**Softening the Tone (B2 Level)**
Imperatives can sound aggressive. Soften them for collaboration:
* Add Courtesy: Please review this when you have a moment.
* Include Yourself: Let's double-check the logs together.

**Noticeable Feedback**
* Drop the Subject: Don't add pronouns before the command.
  * Incorrect: You stop the server! -> Correct: Stop the server.
