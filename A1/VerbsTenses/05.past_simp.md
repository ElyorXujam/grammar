### **Past Simple (Finished Actions)**

**Structure**

1. Regular Verbs
* Add -ed: Walk -> Walked / Optimize -> Optimized

2. Irregular Verbs (No Rules)
* Go -> Went / Have -> Had / See -> Saw

3. Negatives & Questions -> Use Did / Didn't
* Crucial: The main verb resets to the base form.
* Plus: I found the bug.
* Minus: I didn't find the bug. (Not: didn't found)
* Ask: Did you see the log? (Not: Did you saw)

**Noticeable Feedback**
* Double Past Tense: Once did enters, the past tense on the main verb leaves.
  * Incorrect: Where did you went? -> Correct: Where did you go?
